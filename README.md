House Price Prediction is a supervised machine learning project developed to estimate residential property prices based on selected housing features. The project demonstrates an end-to-end regression workflow, including data preprocessing, feature selection, model training, prediction, and evaluation. A Linear Regression model is implemented using Scikit-learn to learn the relationship between housing attributes and sale prices. Performance is assessed using regression metrics such as Mean Absolute Error (MAE), Mean Squared Error (MSE), and Root Mean Squared Error (RMSE). This project provides practical experience in building predictive models for real estate analytics while strengthening fundamental concepts in regression, data analysis, and machine learning.
